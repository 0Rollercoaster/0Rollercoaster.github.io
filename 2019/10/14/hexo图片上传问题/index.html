hexo图片上传问题

2019-10-14 hexo

我的 hexo-asset-image 是 1.0 hexo 3.0

1.先到博客根目录下 查看 _config文件找到 post_asset-folder字段确定post_asset-folder设置为true

2.当我设置参数后，建立文件时，hexo会自动建立一个与文章名同名的文件夹，用来存放跟该文章相关的所有资源

3.我想要把图片更新到仓库的时候总是报错404，index里的路径是 .//com/xxx.jpg这个路径肯定是错误的，找了很多方法，按照一些博客的方法改了hexo-asset-image的index.js，结果是部署后路径发生变化，但是仍然404

4.找了很久终于成功了:

在博客根目录执行 npm install https://github.com/CodeFalling/hexo-asset-image --save 安装插件即可
