Git学习2

2019-12-16 Git

​ 今天在廖雪峰老师的网站上看了Git学习的文档，了解到了很多之前没有接触到的功能，更真切的感受到Git的强大

​ 引用廖老师的说法(时光穿梭机)
​ 版本回退功能 (这是我觉得git很强大的地方!)

​ git log 命令查看自己都干了些啥，可以看到从最近到最远的3次提交
​ 注: 看到的一大串 1347avd...的是 commit id(版本号) ，

​ git reset --hard HEAD^ ^ 代表回退到上一个版本
​ 注:Git在内部有个指向当前版本的HEAD指针.当回退版本的时候,Git只是把HEAD的指向变化,然后顺便把工作区的文件更新

​ 如果我回退到某个版本,关掉了电脑,第二天后悔想恢复到最新版本
​ git reflog 查看我的每一次命令 然后执行 git reset --hard HEAD+ 版本号 就可以坐上时光机飞向未来了

​ 最后记下廖老师的网站:https://www.liaoxuefeng.com/wiki/896043488029600 (学Git都可以看看)
