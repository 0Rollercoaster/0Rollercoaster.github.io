css 由选择器，以及一条或者多条声明组成
例如 ： p{font-size:100px; background:red;}

样式表分为三种
1.内联样式表 （权重最高）
<标签名 style="属性:属性值;"></标签名>

2.内部样式表

3.外部样式表

注：1.外部样式表与内部样式表的权重按照加载顺序
2.@import方式不支持css2之前版本
3.当一个页面被加载的时候（就是被浏览者浏览的时候），link引用的CSS会同时被加载，而@import引用的CSS 会等到页面全部被下载完再被加载。所以有时候浏览@import加载CSS的页面时开始会没有样式。
4.@import是CSS2.1提出的，所以老的浏览器不支持，@import只在IE5以上的才能识别，而link标签无此问题。
