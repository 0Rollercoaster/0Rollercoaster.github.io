面向对象（三大特点：封装、继承、多态）

封装

面向对象：程序中对现实中的一个事物进行封装，把他的属性变为对象的属性，把他的方法变为对象的函数
对象：程序描述一个事物的数据结构
包含：属性和方法

js引用类型都是对象

如何创建自定义对象：3种
1、*直接量方式：
var obj={ “属性名”:属性值, … “方法名”:function(){}, … }

    注：其实属性名和方法名的引号是可以省略的，不推荐
    如何访问对象的属性和方法
        obj.属性名 -- 得到了属性值
            等效于obj["属性名"];
        obj.方法名() -- 调用了此方法;
            等效于obj["方法名"]();
    强烈建议：用.访问对象的属性和方法
    ***js中一切都是对象，一切对象的底层都是hash数组
    特殊：1、访问到不存在的属性，返回undefined
          2、随时随地添加不存在的属性

    ***this的指向：
        1、单个元素绑定事件，this->这个元素
        2、多个元素绑定事件，this->触发的当期那元素
        3、定时器中使用this，this->window
        4、箭头函数使用this，this->window
        5、***函数中的this，this->调用此函数的人
        6、构造函数中的this，指的是正在创建的对象
    何时使用：当前对象的方法内，要访问当前对象自己的东西时，就必须使用this.属性名/方法名()

    何时创建对象：任何时候我们都要优先封装为一个对象，不适合初学者！

2、预定义构造函数：var obj=new Object();
           obj.属性名=属性值;
           obj.方法名=function(){};

3、*使用以上两个方法一次只适合创建一个对象，如果要创建多个：2步
    1、创建一个自定义构造函数
        function h52005(name,age,salary){
            this.name=name;
            this.age=age;
            this.salary=salary;
        }

    2、反复调用自定义构造函数创建对象
        var yxr=new h52005("杨翔睿",18,3500);
        var zs=new h52005("张森",19,4500);

4、遍历对象：可以获取对象中的所有的属性值
    for(var i in 对象名){
        对象名[i];
    }

继承：

父对象的成员（属性和方法），子对象可以直接使用
为什么要继承：代码重用，节约内存空间！
何时继承：只要多个子对象公用的属性或方法，都要集中定义在父对象中

父对象（原型）：保存一类子对象共有属性和共有方法的父对象，所有的子对象都自动继承原型对象
